**Enterprise B2B Dental Photography & DICOM Management Software**

## 🚀 Overview
ClinicLens is a fully developed, production-ready desktop software built specifically for dental clinics. It manages patient clinical photography, renders complex medical DICOM files natively, and generates automated PDF reports. Engineered with a highly scalable, modular architecture, it includes a built-in hardware licensing system, making it immediately ready for distribution and monetization.

## ✨ Premium Enterprise Features

- **🧠 Asynchronous DICOM Engine** Native support for 3D medical scans (MRI/CT). The software extracts and renders slices without memory crashes, running on background threads to keep the UI perfectly fluid.
- **📷 DSLR Camera Watchdog** An automated directory monitor that instantly detects and imports high-res photos transferred via Wi-Fi SD cards from the clinic's reflex camera.
- **🔒 Hardware-Locked Licensing System (DRM)** Built-in DRM generating unique serial keys based on the client's hardware ID. Allows the distribution of lifetime licenses or SaaS-style subscription kill-switches.
- **🛡️ GDPR Compliance Built-in** Automated EULA acceptance tracking, patient privacy consent toggles, and secure AES-encrypted database architecture.
- **⚡ Smart Search & Hotkeys** Global "Spotlight" search (Ctrl+K) with debounce filtering, creating a frictionless, premium user experience.

## 💻 Tech Stack & Architecture
- **Frontend:** CustomTkinter (Modern, modular, MVC-inspired architecture)
- **Backend/Data:** Python, SQLite (LAN Multi-client ready)
- **Medical Imaging:** Pydicom, Pillow (Numpy-accelerated)
- **Packaging:** PyInstaller (Standalone Windows `.exe`)
